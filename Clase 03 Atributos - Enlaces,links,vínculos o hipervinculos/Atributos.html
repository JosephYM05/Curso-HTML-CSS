HTML - Atributos


Los atributos determinan cierta información sobre la etiqueta (o su modo de actuar) y generalmente van asociados a un valor determinado. Este par atributo-valor  se escribe después del nombre de la etiqueta, separándola por espacio y antes del carácter > de la etiqueta de apertura:

class: se utiliza para especificar una o varias clases CSS para un elemento.
id: se utiliza para especificar un identificador único para un elemento.
style: se utiliza para especificar uno o varios estilos CSS para un elemento.
title: se utiliza para proporcionar información adicional sobre un elemento cuando se sitúa el cursor sobre él.
href: se utiliza en los elementos de enlace (< a >) para especificar la URL de destino del enlace.
src: se utiliza en los elementos de imagen (< img >) y en los elementos de audio y vídeo (< audio > y < video >) para especificar la fuente del medio.
alt: se utiliza en los elementos de imagen (< img >) para proporcionar un texto alternativo que se mostrará si la imagen no se puede cargar.
width y height: se utilizan en los elementos de imagen (< img ) y en los elementos de audio y vídeo (< audio > y < video >) para especificar el ancho y la altura del medio.
target: se utiliza en los elementos de enlace (< a> ) para especificar si el enlace debe abrirse en una nueva ventana o pestaña del navegador.
rel: se utiliza en los elementos de enlace (< a >) para especificar la relación entre la página actual y la página de destino del enlace (por ejemplo, rel="nofollow" indica que los motores de búsqueda no deben seguir el enlace). type: se utiliza en los elementos de formulario (< input >, < button >, < select>, etc.) para especificar el tipo de control de formulario.
